# Fixture Vocabulary

[Back to Figma AI Agent Daemon Boundary](../figma-ai-agent-daemon-boundary.md)

Figma and older discussions used "template" loosely, but current SSOT has no
daemon-owned agent template entity.

`리도`, `영실`, `홍도`, and `지원` are server-provided fixtures. Selecting one creates
a normal upstream agent. Daemon never hardcodes fixture catalog, descriptions, or
instruction copy; it only receives the assignment-time instruction/runtime/model
snapshot.
